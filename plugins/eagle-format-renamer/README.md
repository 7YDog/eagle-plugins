# 格式重命名

Eagle 窗口插件，用于对当前选中的一个或多个项目按规则重命名。

## 规则

- `%F` 当前项目的第一个文件夹名称
- `%D` 日期，可在窗口里选择日期来源和格式
- `%N` 序号，可设置起始值、递增和补零位数

示例：`%F-发布%N` 会生成类似 `7月6日-发布1` 的名称。

## 使用

1. 在 Eagle 里选中一个或多个文件。
2. 打开插件“格式重命名”。
3. 输入或选择历史规则。
4. 确认预览无误后点击“执行重命名”。

插件会把使用过的规则保存在本机 Eagle 插件窗口的本地记录中，方便下次直接复用。

## 记录与刷新

- 常用记录只显示最近 5 条，点击整张卡片即可选用。
- 插件窗口打开后会自动检测 Eagle 当前选中项变化，并同步刷新预览。
- 执行、刷新、选用历史规则等记录会写入插件本地日志和控制台。
